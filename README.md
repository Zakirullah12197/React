# 🚀 My React Mini Projects

This repository contains my **React practice projects** — small apps built while learning React step by step.  
Each project explores **React concepts, hooks, routing, context, and Redux Toolkit**.  

---

## 🗂 Projects

- **01_myreactapp** – First React app, exploring components & JSX  
- **02_CounterHookes** – Counter app using `useState`  
- **03_TaskManager** – Simple Todo / Task manager  
- **04_bgChanger** – Change background dynamically  
- **05_PasswordGenerator** – Random password generator  
- **06_CurrencyChanger** – Currency converter  
- **07_reactRouter** – Multi-page routing practice  
- **08_MiniContext** – Sharing state with Context API  
- **09_ThemeSwitcher** – Dark/Light theme toggler  
- **10_TodoContextLocal** – Todo app with Context + Local Storage  
- **11_ReduxToolkitTodo** – Todo app using Redux Toolkit  
- **12_MegaBlogProject** – Bigger project combining multiple concepts

`useState` | `useEffect` | `useReducer` | `useContext` | React Router | Redux Toolkit | Forms | Local Storage
